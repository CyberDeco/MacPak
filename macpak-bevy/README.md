# Controls:

Auto-centering:
  - Camera automatically fits to model bounds when loaded
  - Starts at a nice 3/4 view angle (45Â°)
  - Zoom limits scale with model size

Keyboard controls:
  - R - Reset camera to default view
  - W - Toggle wireframe mode (lime green)
  - G - Toggle ground grid
  - B - Toggle skeleton

Ground grid:
  - Semi-transparent 10x10 grid at origin
  - Helps with scale/orientation

Panning:
  - Middle mouse drag: Orbit
  - Shift + Middle mouse drag: Pan
    - Arrow keys
    - Shift + right-drag
  - Scroll wheel: Zoom
  - Trackpad pinch: zoom
  - Trackpad two-finger rotate: orbit
